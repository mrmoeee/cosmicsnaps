# Frontend routes
Our components are organized as follows

* `Root`
  *  `App`
    * `Navbar`
    * `Main components`
    * `Footer`

Main components
* `/`
  * `Home Feed`
* `/signin`
  * `SessionForm`
* `/sigup`
  * `SessionForm`
* `/photos/new`
  * `UploadPhotoForm`
* `/photos/:p_id`

  * `CommentIndex`
  * `CommentIndexItems`
  * `UserIndex`
  * `UserIndexItems`
* `/users/:u_id`

  * `InformationIndex`
  * `InformationIndexItems`
